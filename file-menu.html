File Menu
Project loading/saving, imports, exports, and preset tools in Bambu Studio 2.4.0.70.
Bambu Studio: 2.4.0.70 Verified on: 2025-11-28 Profile: A1 · No AMS
Overview
The File menu controls projects, importing models, exporting sliced files, and preset backup – but never talks directly to the printer.
  • New Project (Ctrl+N) – clear the plate and start from scratch.
  • Open Project… (Ctrl+O) – load a saved 3MF, replacing the current one.
  • Recent Projects ▶ – quick access to recently opened 3MFs.
  • Save Project (Ctrl+S) – save to existing 3MF (greyed out until named).
  • Save Project as… (Ctrl+Shift+S) – save a new 3MF copy.
Import
  • Import 3MF/STL/STEP/SVG/OBJ/AMF… (Ctrl+I) – add geometry to the current plate.
  • Import Configs… – bring in profiles from another system or backup.
Export
  • Export all objects as one STL… – merge everything to a single STL.
  • Export all objects as STLs… – each object becomes a separate STL.
  • Export Generic 3MF… – 3MF usable in other slicers.
  • Export plate sliced file… – sliced file for the current plate.
  • Export all plate sliced file… – sliced files for all plates.
  • Export G-code… – raw G-code output.
  • Export toolpaths as OBJ… – toolpath visualization model.
  • Export Preset Bundle… – backup of your printer/filament/process profiles.
  • Publish to MakerWorld – upload models/plates to MakerWorld.
  • Batch Preset Management – bulk edit and clean up presets.
  • Quit – exit Bambu Studio (prompts to save if needed).
Workflow tips
  • Use Save Project as… before you start heavily editing a plate.
  • Use Import to add new parts without destroying your layout.
  • Use Export all plate sliced file… for multi-plate jobs.
  • Back up tuned A1 profiles with Export Preset Bundle….
