Preferences

How Bambu Studio itself behaves. These settings affect the UI and workflows, not printer firmware.

Preferences Overview

Preferences control Studio behavior, rendering, storage locations, and update handling. They do not directly change print parameters.

General

  • Units: Keep in millimeters.
  • Clear "Don't show again": Restores suppressed dialogs.
  • Transfer modified values: Controls whether tweaks persist when switching presets.

3D Settings

Viewer-only options. These affect navigation, performance, and visuals only.

  • Zoom to mouse position
  • Level-of-detail rendering
  • Advanced G-code preview

Presets & System Preset Updates

System preset updates modify built-in templates (purge lines, start/end behavior, plate parameters). These updates can change printer motion without a firmware update.

File Associations

  • 3MF – recommended
  • STL – optional
  • STEP – useful for CAD imports

Projects, Downloads & Media

Controls autosave, backup behavior, and download locations.

Theme & Developer Mode

Developer mode enables experimental features. Leave disabled unless explicitly required.
